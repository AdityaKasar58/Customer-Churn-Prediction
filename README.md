# Customer Churn Prediction

## Project Overview
Predicted customer churn for a telecom company
using Machine Learning models on 7,000+ records.

## Tools Used
- Python (Pandas, Scikit-learn, Seaborn)
- Machine Learning (Random Forest, Decision Tree)
- Tableau Public
- Jupyter Notebook

## Key Findings
- Month-to-month customers 2.5x more likely to churn
- Random Forest achieved 85% accuracy
- F1-Score: 0.83
- Top churn factors: Contract type, Tenure, Monthly charges

## Dataset
https://www.kaggle.com/datasets/blastchar/telco-customer-churn
